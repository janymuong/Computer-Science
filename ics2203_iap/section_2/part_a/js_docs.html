Introduction

JavaScript is an object-oriented scripting language...

It is widely used for creating dynamic content on the web, interacting with the DOM, and building robust web applications.

JavaScript is an object-oriented scripting language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.

Prerequisite

You should be familiar with the following:

  • A general understanding of the Internet.
  • Working knowledge of HyperText Markup Language (HTML).
  • Some programming experience, preferably with a language like JavaScript, Python, or Java.

Hello World

As a tradition, write a "Hello world" JavaScript code:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

This simple example demonstrates the use of a basic JavaScript function to display an alert message.

Variables & Declaration

Variables hold values in your application...

JavaScript identifiers must follow certain rules, including starting with a letter, underscore, or dollar sign.

Three ways to declare a variable:

  • Using the keyword const: const x = 42;
  • Directly assigning a value: x = 42; (Avoid this for global variables)
  • Using the keyword let: let y = 13; (Block scope local variable)

Constants

You can create a read-only, named constant with the const keyword...

A constant cannot change value through assignment or be re-declared while the script is running.

Example:

const PI = 3.14;

Constants must be initialized with a value and follow the same rules as variables for naming.

Data Types

The ECMAScript standard defines seven data types:

  • Six primitive data types:
    • Boolean: true and false
    • Null: A special keyword denoting a null value
    • Undefined: A top-level property whose value is undefined
    • Number: 42 or 3.14159
    • String: "Howdy"
    • Symbol (new in ECMAScript 2015): A unique and immutable data type
  • Object

Objects and functions are fundamental elements in JavaScript, serving as containers for values and procedures.

Functions

A function definition (also called a function declaration)...

Example:

function square(number) { return number * number; }

Functions consist of the function keyword, a name, a list of arguments, and the statements enclosed in curly brackets.

Parameters are passed by value, meaning changes inside the function do not affect the global or calling function.
