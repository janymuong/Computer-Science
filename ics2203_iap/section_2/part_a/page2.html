JavaScript Functions - calling() them :)

As from the introductory information you must define a function somewhere first in the scope from which you wish to call it.

Here's a simple example:

function square(number) {
  return number * number;
}

You can call this function like this:

const result = square(5);
console.log(result); // Outputs 25
